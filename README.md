Codename WordBridge
===================

**Modular chat for distributed team collaboration**


USAGE
-----

1. Install [Node.js](http://nodejs.org/)
2. Download / Clone this git repository
3. `npm install`
4. `node app.js`
5. open `http://localhost:3000`



RANDOM FACTS
------------

* Alpha status.
* IRC backend.
